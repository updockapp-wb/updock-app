CREATE TABLE sessions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  spot_id UUID NOT NULL REFERENCES spots(id) ON DELETE CASCADE,

export interface Session {

2. **Define `SessionsContextType` interface:**
```typescript
interface SessionsContextType {
  sessions: Session[];
  isLoadingSessions: boolean;
  fetchSessionsForSpot: (spotId: string) => Promise<void>;
  createSession: (spotId: string, startsAt: string, note?: string) => Promise<void>;
  joinSession: (sessionId: string) => Promise<void>;
  leaveSession: (sessionId: string) => Promise<void>;
  cancelSession: (sessionId: string) => Promise<void>;
  userUpcomingSessions: (Session & { spot_name: string })[];
  fetchUserSessions: () => Promise<void>;
  isLoadingUserSessions: boolean;
}
```

3. **Create `SessionsProvider` component:**
   - `const { user } = useAuth();`
   - State: `sessions` (Session[]), `isLoadingSessions` (boolean), `userUpcomingSessions` ((Session & { spot_name: string })[]), `isLoadingUserSessions` (boolean)

4. **`fetchSessionsForSpot(spotId)`:**
   - Query: `supabase.from('sessions').select('*, session_attendees(user_id)').eq('spot_id', spotId).eq('is_cancelled', false).gte('starts_at', new Date(Date.now() - 60 * 60 * 1000).toISOString()).order('starts_at', { ascending: true })`
   - Collect all unique `creator_id` values from results
   - Batch fetch profiles: `supabase.from('profiles').select('id, display_name, avatar_url, avatar_id').in('id', creatorIds)`
   - Build profile map: `Record<string, { display_name, avatar_url, avatar_id }>`
   - Map results to `Session[]`: set `creator_profile` from map, `attendee_count = row.session_attendees.length`, `user_is_attending = row.session_attendees.some(a => a.user_id === user?.id)`

5. **`createSession(spotId, startsAt, note?)`:**
   - Insert into `sessions`: `{ spot_id: spotId, creator_id: user.id, starts_at: startsAt, note: note?.trim() || null }`
   - Use `.select('*').single()` to get the created session back
   - Then insert creator into `session_attendees`: `{ session_id: data.id, user_id: user.id }` (per CONTEXT.md decision: creator auto-inserted as attendee)
   - After both inserts succeed, refetch sessions for the spot: call `fetchSessionsForSpot(spotId)` to get fresh data with profile join

6. **`joinSession(sessionId)`:**
   - Optimistic update: `setSessions(prev => prev.map(s => s.id === sessionId ? { ...s, attendee_count: s.attendee_count + 1, user_is_attending: true } : s))`
   - DB: `supabase.from('session_attendees').upsert({ session_id: sessionId, user_id: user.id }, { onConflict: 'session_id,user_id' })`
   - On error: rollback optimistic update (decrement count, set user_is_attending false)

7. **`leaveSession(sessionId)`:**
   - Optimistic update: decrement attendee_count, set user_is_attending false
   - DB: `supabase.from('session_attendees').delete().eq('session_id', sessionId).eq('user_id', user.id)`
   - On error: rollback (increment count, set user_is_attending true)

8. **`cancelSession(sessionId)`:**
   - Optimistic update: `setSessions(prev => prev.filter(s => s.id !== sessionId))`
   - DB: `supabase.from('sessions').update({ is_cancelled: true }).eq('id', sessionId).eq('creator_id', user.id)`
   - On error: refetch sessions for the spot (stored spotId from sessions state)

9. **`fetchUserSessions()`:**
   - Query: `supabase.from('session_attendees').select('session_id, sessions(*, spots(name))').eq('user_id', user.id).gte('sessions.starts_at', new Date(Date.now() - 60 * 60 * 1000).toISOString())` — NOTE: if Supabase embedded filter on nested relation doesn't work cleanly, use a two-step approach: first fetch session_attendees for user, then fetch sessions by IDs with spots join, then filter client-side by starts_at >= now - 1h
   - Alternative simpler approach: `supabase.from('sessions').select('*, session_attendees!inner(user_id), spots(name)').eq('session_attendees.user_id', user.id).eq('is_cancelled', false).gte('starts_at', new Date(Date.now() - 60 * 60 * 1000).toISOString()).order('starts_at', { ascending: true }).limit(3)`
   - Map to `(Session & { spot_name: string })[]` where `spot_name = row.spots.name`
