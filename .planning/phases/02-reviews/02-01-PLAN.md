**Create `src/components/ReviewForm.tsx`:**

Define a `Review` interface at the top of the file (to be used by both components):
```typescript
export interface Review {
  id: string;
  spot_id: string;
  user_id: string;
  rating: number;
  comment: string | null;
  created_at: string;
  updated_at: string;
  profiles: {
    display_name: string | null;
    avatar_url: string | null;
    avatar_id: number | null;
  } | null;
}
```

Props interface:
```typescript
interface ReviewFormProps {
  spotId: string;
  userReview: Review | null;
  onSubmit: (review: Review) => void;
  onDelete: () => void;
}
```

Component behavior:
- Accept `spotId`, `userReview` (existing review to edit or null for new), `onSubmit` callback, `onDelete` callback
- Local state: `rating` (number, 0 = unset), `comment` (string), `isSubmitting` (boolean), `error` (string | null), `isEditing` (boolean — true when user clicks "Edit" on their existing review)
- If `userReview` is non-null AND `isEditing` is false, show a read-only summary of their review with an "Edit" button (`t('review.edit')`) and a "Delete" button (`t('review.delete')`)
- If `userReview` is null OR `isEditing` is true, show the form:
  - Star picker: 5 `<button>` elements using `Star` from `lucide-react`. Filled star: `className="fill-amber-400 text-amber-400"`. Empty star: `className="text-slate-300 hover:text-amber-300"`. Size 28.
  - Comment textarea: `<textarea>` with `maxLength={1000}`, `minLength={20}`, placeholder `t('review.placeholder')`, className `w-full p-3 border border-slate-200 rounded-xl text-sm resize-none focus:outline-none focus:ring-2 focus:ring-sky-500/30 focus:border-sky-500 bg-white`, 4 rows
  - Character count display: `{comment.length}/1000` in `text-xs text-slate-400`
  - Submit button: disabled when `rating === 0 || comment.trim().length < 20 || isSubmitting`. Text: `t('review.submit')` (or `t('review.update')` if editing). Styled: `w-full py-3 bg-gradient-to-r from-sky-500 to-blue-600 text-white font-bold rounded-xl disabled:opacity-50`
  - Error display: `{error && <p className="text-red-500 text-sm mt-2">{error}</p>}` — NO alert() calls
- When `isEditing` becomes true, pre-fill `rating` and `comment` from `userReview`
- On submit:
  1. Set `isSubmitting = true`, clear error
  2. Call `supabase.from('reviews').upsert({ spot_id: spotId, user_id: user.id, rating, comment, updated_at: new Date().toISOString() }, { onConflict: 'spot_id,user_id' }).select('*, profiles(display_name, avatar_url, avatar_id)').single()`
  3. On success: call `onSubmit(data)`, reset `isEditing` to false
  4. On error: set `error` to the error message string
  5. Set `isSubmitting = false`
- On delete:
  1. Call `supabase.from('reviews').delete().eq('spot_id', spotId).eq('user_id', user.id)`
  2. On success: call `onDelete()`
  3. On error: set `error` to error message
