```typescript
interface ProfileContextType {
  profile: ProfileData | null;
  isLoading: boolean;

  uploadAvatar: (file: File) => Promise<void>;
  selectPresetAvatar: (avatarId: number) => Promise<void>;
}
```

**Context + Provider:**

```typescript
const ProfileContext = createContext<ProfileContextType | undefined>(undefined);

```

**Provider implementation details:**
1. State: `profile` (ProfileData | null), `isLoading` (boolean)
2. `useEffect` on `user` change: if user exists, fetch profile from `supabase.from('profiles').select('display_name, avatar_url, avatar_id').eq('id', user.id).single()`. Set profile state. On user null, set profile to null.
3. `updateDisplayName(name: string)`: call `supabase.from('profiles').upsert({ id: user.id, display_name: name })`. On success, update local profile state optimistically.
4. `uploadAvatar(file: File)`:
   - Extract file extension: `file.name.split('.').pop()`
   - Build path: `` `${user.id}/avatar.${fileExt}` ``
   - Upload: `supabase.storage.from('avatars').upload(filePath, file, { upsert: true })`
   - Get public URL: `supabase.storage.from('avatars').getPublicUrl(filePath)`
   - Save URL: `supabase.from('profiles').upsert({ id: user.id, avatar_url: publicUrl })`
   - Update local profile state with new avatar_url
   - On upload error, console.error and throw (let caller handle UI feedback)
5. `selectPresetAvatar(avatarId: number)`: call `supabase.from('profiles').upsert({ id: user.id, avatar_id: avatarId, avatar_url: null })`. Clear avatar_url when switching to preset. Update local state.

```typescript
  }

```
